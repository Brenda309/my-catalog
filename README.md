## Getting Started
To get a local copy up and running follow these simple example steps.


### Prerequisites
Ruby 3.1.2 or higher
RSpec 3.11.0
### Setup
    cd  git clone git@github.com:Brenda309/my-catalog.git
    cd my-catalog/

### Install
     bundle install

     gem install rspec
### Usage

### Run tests
      rspec spec
